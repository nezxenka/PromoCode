## Технологии

### Основные
- Java 17
- Paper/Spigot API 1.16.5
- Maven 3.x

### Библиотеки
- Lombok 1.18.30 (compile-time)
- HikariCP 5.0.1 (connection pooling)
- PlaceholderAPI 2.11.2 (optional)

### Паттерны проектирования
- Builder Pattern
- Service Pattern
- Repository Pattern
- Observer Pattern
- Singleton Pattern

## Возможности

1. **Система промокодов**
   - Активация через команды
   - Ограничения использований
   - Группировка промокодов

2. **База данных**
   - MySQL с HikariCP
   - Асинхронная обработка
   - Оптимизированные запросы

3. **Управление**
   - Команды администратора
   - Статистика активаций
   - Информация о промокодах

4. **Интеграции**
   - LinkAlert (привязка аккаунтов)
   - PlaceholderAPI (опционально)

5. **Локализация**
   - Полная система сообщений
   - Легкий перевод через messages.yml

## Команды

```
/promocode reload - Перезагрузить конфигурацию
/promocode info <код> - Информация о промокоде
/promocode stats - Статистика активаций
/<промокод> - Активировать промокод
```

## Сборка

```bash
mvn clean package
```

Результат: `target/PromoCode-1.0-SNAPSHOT.jar`

## Требования

### Сервер
- Paper/Spigot 1.16.5 или выше
- Java 17 или выше
- MySQL 5.7+ или MariaDB 10.2+

### Плагины
- LinkAlert (опционально, для привязки аккаунтов)
- PlaceholderAPI (опционально)
