## Возможности

- ✅ Система промокодов с активацией через команды
- ✅ Поддержка MySQL с HikariCP пулом соединений
- ✅ Ограничение использований на игрока и глобально
- ✅ Группировка промокодов
- ✅ Интеграция с LinkAlert для привязки аккаунтов
- ✅ Асинхронная обработка активаций
- ✅ Полная кастомизация сообщений
- ✅ Статистика активаций
- ✅ Современная архитектура с использованием Lombok

## Требования

- Java 17+
- Spigot/Paper 1.16.5
- MySQL 5.7+ или MariaDB 10.2+

## Установка

1. Скомпилируйте плагин: `mvn clean package`
2. Скопируйте `PromoCode-1.0-SNAPSHOT.jar` в папку `plugins`
3. Настройте `database.yml` с вашими данными MySQL
4. Перезапустите сервер

## Конфигурация

### config.yml
Основная конфигурация с промокодами

### database.yml
Настройки подключения к базе данных

### messages.yml
Все сообщения плагина

## Команды

- `/promocode reload` - Перезагрузить конфигурацию
- `/promocode info <код>` - Информация о промокоде
- `/promocode stats` - Статистика активаций
- `/<промокод>` - Активировать промокод

## Права

- `promocode.admin` - Доступ к административным командам
